# ProjectEuler - Rust
An attempt to learn Rust.lang by completing the [ProjectEuler.net](https://projecteuler.net) problems.

Take this with a grain of salt.

## Requirements
 - Rust.lang
 - Cargo

## Installation & Usage
```
git clone https://github.com/johanleroux/projecteuler.git
cargo run
```
